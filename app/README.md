# 淘宝种草 AI 全链路协作平台

## 项目简介

这是一个服务于淘宝种草全链路的 AI 协作平台，目前优先支持**下行内容团队**和**直管执行团队**使用。

## 4 月上线功能

- Brief 智能评估
- 智能选号与达人分发
- AI 内容审核与修正
- 发布&投流监测
- 初步内部复盘

## 快速开始

### 本地开发

直接打开 `index.html` 文件即可在浏览器中运行。

### 部署到 Vercel

1. Fork 或克隆此仓库
2. 在 Vercel 导入项目
3. 自动部署完成

### 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库 Settings > Pages 中启用 GitHub Pages
3. 选择 main 分支和 /root 目录

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- localStorage 数据持久化
- 零后端依赖

## 数据说明

所有数据存储在浏览器 localStorage 中。清除浏览器数据会导致数据丢失。

5 月版本将接入后端数据库实现多用户数据隔离。
