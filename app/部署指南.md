# 淘宝种草AI协作平台 - 部署指南

## 快速部署到Vercel（推荐）

### 步骤1：创建GitHub仓库

1. 访问 https://github.com/new
2. 仓库名称：`taobao-zhongcao-ai`
3. 设置为Public（公开）
4. 不要勾选"Add a README file"
5. 点击"Create repository"

### 步骤2：推送代码到GitHub

在命令行中执行以下命令（替换YOUR_USERNAME为你的GitHub用户名）：

```bash
cd "C:\Users\高有才\.qoderwork\workspace\mncv91pk7f82ff12"
git remote add origin https://github.com/YOUR_USERNAME/taobao-zhongcao-ai.git
git branch -M main
git push -u origin main
```

### 步骤3：部署到Vercel

1. 访问 https://vercel.com/new
2. 点击"Import Git Repository"
3. 选择刚才创建的 `taobao-zhongcao-ai` 仓库
4. 保持默认配置（Vercel会自动识别vercel.json）
5. 点击"Deploy"
6. 等待1-2分钟，部署完成后会生成网址

### 步骤4：获取访问网址

部署成功后，你会获得类似这样的网址：
```
https://taobao-zhongcao-ai.vercel.app
```

把这个网址分享给团队即可使用！

## 常见问题

**Q: 部署后无法访问？**
A: 检查Vercel/Netlify的部署日志，确保没有错误

**Q: 如何更新代码？**
A: 修改本地文件后，执行：
```bash
git add .
git commit -m "更新说明"
git push
```
Vercel/Netlify会自动重新部署

**Q: 如何设置自定义域名？**
A: 在Vercel/Netlify的项目设置中添加自定义域名（需要你有域名）

**Q: 数据会丢失吗？**
A: 4月版本数据存在浏览器localStorage中，只要不清缓存就不会丢失。但不同设备/浏览器之间数据不共享。
