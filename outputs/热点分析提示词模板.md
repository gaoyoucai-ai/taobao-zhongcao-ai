# 小红书热点分析提示词模板

## 1. 热点监控提示词

```
你是一位专业的小红书种草策略分析师。请分析以下热点信息，提取适合种草的选题：

【输入数据】
{热点数据}

【分析维度】
1. 热点类型：美妆/穿搭/家居/数码/食品/生活方式
2. 热度评估：高/中/低
3. 种草潜力：是否适合产品植入
4. 目标人群：年龄段、性别、消费能力
5. 内容角度：测评/教程/对比/清单/场景

【输出格式】
## 今日热点选题推荐（Top 5）

### 选题1：[标题]
- 热度等级：⭐⭐⭐⭐⭐
- 目标人群：
- 内容角度：
- 种草切入点：
- 预计效果：

### 选题2-5...

## 避坑提醒
- 不适合种草的热点：
- 需要谨慎跟进的敏感话题：
```

## 2. 爆款案例分析提示词

```
分析以下小红书爆款笔记，提取可复制的方法论：

【输入数据】
{爆款笔记数据}

【分析框架】
1. 标题公式：提取标题结构
2. 封面逻辑：视觉元素分析
3. 内容结构：开头-中间-结尾
4. 种草技巧：产品植入方式
5. 互动设计：如何引导评论/收藏

【输出】
- 爆款公式总结
- 可套用的模板
- 适配我产品的改编建议
```

## 3. 竞品监控提示词

```
监控竞品{竞品名称}的最新种草动态：

【分析内容】
1. 近期投放达人清单
2. 主推产品/卖点
3. 内容形式偏好
4. 投放节奏
5. 爆款内容特征

【输出】
- 竞品策略总结
- 差异化机会点
- 我方应对建议
```

```
基于以下数据，预测下周种草趋势：

【输出】
## 下周趋势预测

### 趋势1：[趋势名称]
- 判断依据：
- 持续时间预估：
- 种草机会：
- 建议动作：

### 趋势2-3...
```
