```
│  8.验收标准 → CPUV 分级和是否追加                  │
└────────────────────────────────────────────────────┘
```

## 📝 内容结构模板（万能公式）

```
【开头】0-30 秒
目标：抓住注意力，引发共鸣
必须包含：痛点场景 + 问题引入
禁止：硬广开场、念参数
参考话术："你有没有遇到过_______的情况？"

【中段】30 秒-2 分钟
目标：建立信任，展示价值
必须包含：
  ✓ 核心卖点 1 展示（证据 + 场景）
  ✓ 核心卖点 2 展示（对比 + 数据）
  ✓ 核心卖点 3 展示（体验 + 感受）
禁止：纯念参数、无场景、过度美化
参考话术："我用了 X 周，最大的感受是_______"

【结尾】2 分钟 - 结束
目标：促成行动
必须包含：
  ✓ 购买引导（点击哪里）
  ✓ 优惠信息（限时/限量）
  ✓ 评论区互动（置顶什么）
禁止：只说"去买"，不给理由
参考话术："如果_______，真的可以试试这个"
```

## 📊 CPUV 分级标准（参考）

| 类目 | 优秀 | 良好 | 一般 | 较弱 | 差 |
|-----|------|------|------|------|----|
| **美妆护肤** | ≤8 | 8-12 | 12-18 | 18-25 | >25 |
| **个护清洁** | ≤6 | 6-10 | 10-15 | 15-20 | >20 |
| **服饰鞋包** | ≤10 | 10-15 | 15-22 | 22-30 | >30 |
| **食品饮料** | ≤5 | 5-8 | 8-12 | 12-18 | >18 |
| **家电数码** | ≤12 | 12-18 | 18-25 | 25-35 | >35 |
| **家居家装** | ≤8 | 8-13 | 13-20 | 20-28 | >28 |
| **母婴用品** | ≤10 | 10-16 | 16-24 | 24-32 | >32 |

**决策规则**：
- CPUV ≤ 优秀：立即追加，签长期
- CPUV 良好：观察至 15 天
- CPUV 一般：不再追加，自然长尾
- CPUV 较差/差：止损，复盘原因

## 🚫 常见避坑指南

### ❌ 人群定义过宽
```
错误：18-45 岁，全国，不限职业
正确：25-32 岁，新一线/二线，职场白领
```

### ❌ 卖点太多太杂
```
错误：列出 10 个卖点，全是重点
正确：聚焦 3 个核心卖点，每个有证据
```

### ❌ 内容要求模糊
```
错误：要有趣、有创意、吸引人
正确：前 30 秒出现产品，必须有对比，展示真实场景
```

### ❌ 验收标准缺失
```
错误：效果好就追加
正确：CPUV≤10 元追加，10-15 元观察，>15 元止损
```

## 💡 高质量 Brief 的四个特征

```
✓ 具体：人群画像具体到年龄段、城市、职业
✓ 量化：KPI 有具体数字，不是"效果好"
✓ 可执行：达人看完知道怎么拍，不是猜谜
✓ 可验收：有明确验收标准，不是主观判断
```

## 🔑 一句话总结

**Brief 的质量 = 投放效果的上限**

**想清楚再投放：**
1. 人群想得越清楚，达人选得越精准

3. 要求写得越具体，内容跑得越不偏
