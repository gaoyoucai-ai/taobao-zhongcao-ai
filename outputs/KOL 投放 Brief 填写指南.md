# KOL 投放 Brief 填写指南

> **适用对象**：品牌方/商家运营人员  
> **使用场景**：KOL 投放前策略梳理和需求明确  
> **预计耗时**：60-90 分钟

---

## 一、为什么需要认真填写这份 Brief？

### Brief 的质量直接影响投放效果

根据 kol-analysis-2.0 方法论对数百个 KOL 投放案例的分析，**Brief 清晰度与投放效果呈强正相关**：

| Brief 质量 | 平均 CPUV | 达标率 | 达人配合度 |
|-----------|----------|--------|-----------|
| **优秀**（完成度≥90%） | 8-12 元 | 85%+ | 高 |
| **良好**（完成度 70-89%） | 12-18 元 | 65-85% | 中 |
| **较差**（完成度<70%） | 18-30 元 | <65% | 低 |

### 这份 Brief 能帮你解决什么问题？

✅ **想清楚再投放**：避免"为了投而投"，明确目标和预期  
✅ **精准选达人**：减少"凭感觉选人"，基于人群匹配度决策  
✅ **内容不跑偏**：达人准确理解需求，减少修改成本  
✅ **验收有标准**：效果评估有据可依，避免扯皮  
✅ **经验可沉淀**：每次投放都是学习，持续优化策略

## 三、核心模块填写详解

### 3.1 第三部分：目标人群定义 ⭐（最重要，建议用时 30 分钟）

这是整份 Brief 的**灵魂**，直接决定达人筛选和内容策略的准确性。

#### 第一步：基础人群画像（必填）

**不要这样填**：
```
年龄：18-40 岁  ❌（范围太宽，等于没说）
性别：女性      ❌（过于笼统）
地域：全国一二线城市 ❌（不够具体）
```

**应该这样填**：
```
年龄：25-32 岁（核心 28-30 岁）✅
性别：女性占比 70-80% ✅
地域：新一线/二线城市为主（成都、杭州、武汉、南京）✅
职业：职场新人至资深白领（工作 1-8 年）✅
收入：月可支配 5000-15000 元，中等消费能力✅
```

**填写技巧**：
- 参考天猫后台"访客分析"中的真实数据
- 如果是新品，参考竞品的人群画像
- 宁可收窄范围，不要过度宽泛

#### 第二步：人群深层洞察（强烈建议填写）

**人群代号**：给目标人群起个名字，方便团队内部沟通

**示例**：
```
人群代号："精致穷的小镇青年阿强"
当前人生阶段：事业上升期（工作 3-5 年）
生活方式关键词：健身党、咖啡控、成分党、国潮爱好者
日常节奏：
  - 工作日：996，早晚通勤各 1 小时，晚上健身 1 小时
  - 周末：宅家补觉/朋友聚餐/偶尔周边游
  - 产品使用时段：早晨出门前、晚上健身后、周末宅家时
```

**5 Why 挖掘深层动机**（以洗发水为例）：

```
Q1: 为什么买控油洗发水？
A: 因为头发容易油

Q2: 为什么怕头发油？
A: 因为显得不精神

Q3: 为什么不精神不行？
A: 因为刚入职，想给同事留好印象

Q4: 为什么在乎同事印象？
A: 因为想快速融入团队，不被当成学生

Q5: 这反映了什么深层需求？
A: 职场身份认同焦虑 → 希望被认可为"专业人士"

→ 种草切入点：不是卖"控油"，而是卖"职场形象管理"
```

#### 第三步：需求层次分析（必填）

**功能需求**：
```
用户遇到的具体问题：头发隔天就油，需要每天洗头
尝试过的解决方案：换过清扬、海飞丝、卡诗，效果不明显
现有方案的不足：要么太干涩，要么留香短，要么性价比低
我们的产品如何解决：专利控油成分，72 小时持久，pH 值温和
```

**情绪需求**：
```
使用后希望获得的感觉：自信、清爽、掌控感
具体描述： "不用担心突然出油尴尬，随时保持最佳状态"
```

**社交需求**：
```
希望在他人眼中呈现的形象：专业、干净、有品味
害怕被如何看待：邋遢、学生气、不修边幅
产品能帮助获得的社交价值：近距离社交也不慌，开会发言更有底气
```

#### 第四步：痛点与恐惧（必填）

**核心痛点排序**：
```
1. 头发油得快（最痛，每天都要面对）
2. 头皮痒但有头屑（社交场合尴尬）
3. 洗完头发干涩打结（体验差）
```

**深层恐惧**：
```
社交恐惧：怕靠近别人时被闻到异味/看到头屑
形象恐惧：怕显得不专业，影响职业发展
决策恐惧：怕花大价钱买了没效果的洗发水
```

**营销启示**：
- 不要只说"控油 72 小时"，要说"重要会议前不慌"
- 不要只说"去屑止痒"，要说"近距离社交也自信"
- 不要只说"大牌成分"，要说"无效退款，真人实测"

#### 第五步：人群内容偏好（必填）

**不要凭感觉猜**，参考以下数据源：
- 天猫后台"内容偏好"数据
- 抖音/小红书后台粉丝画像
- 评论区高频词汇分析
- 问卷调研结果

**示例**：
```
喜欢的内容形式：专业测评、真实体验分享、知识科普
喜欢的内容时长：中视频（3-8 分钟）
喜欢的内容风格：真实接地气型、专业严谨型
讨厌的内容类型：硬广植入太明显、过度美化不真实、纯念参数无场景
```

---

### 3.2 第五部分：内容要求（建议用时 20 分钟）

#### 推荐内容形式（按优先级）

**不要只写"测评视频"**，要具体到场景和结构：

**示例**：
```
形式 1（强烈推荐）：专业测评对比
适用场景：新房装修选灯具、数码产品选购、护肤品成分对比
预期效果：建立专业信任感，驱动理性决策
标杆案例链接：https://bilibili.com/video/BVxxxxx（数玩工场 - 雷士照明）
匹配人群：理性分析型消费者，重视参数和数据

脚本结构建议：
【0-30 秒】痛点引入：新房装修，灯光怎么选才不踩坑？
【30-90 秒】产品亮相 + 核心参数：Ra>90、RG0 无频闪、色温可调
【90-180 秒】安装过程 + 场景演示：实拍安装、不同模式对比
【180-240 秒】使用体验总结：优缺点并陈（增加可信度）
【240-319 秒】购买建议 + 优惠信息：活动价、适合人群
```

#### 禁止的内容形式（明确红线）

**示例**：
```
形式 1：纯氛围展示视频
禁止原因：只拍温馨画面，不讲功能参数，用户看完不知道产品好在哪里
反面案例：某生活美学博主视频，播放 2.3 万，0 转化

形式 2：剧情演绎过度
禁止原因：产品完全沦为道具，用户只记得剧情不记得产品
反面案例：某剧情号视频，播放 9.8 万，CPUV 高达 156 元
```

### 3.3 第八部分：验收标准（建议用时 10 分钟）

#### CPUV 分级标准（参考行业基准）

不同类目 CPUV 差异较大，请参考以下标准：

| 类目 | 优秀 | 良好 | 一般 | 较弱 | 差 |
|-----|------|------|------|------|----|
| **美妆护肤** | ≤8 元 | 8-12 元 | 12-18 元 | 18-25 元 | >25 元 |
| **个护清洁** | ≤6 元 | 6-10 元 | 10-15 元 | 15-20 元 | >20 元 |
| **服饰鞋包** | ≤10 元 | 10-15 元 | 15-22 元 | 22-30 元 | >30 元 |
| **食品饮料** | ≤5 元 | 5-8 元 | 8-12 元 | 12-18 元 | >18 元 |
| **家电数码** | ≤12 元 | 12-18 元 | 18-25 元 | 25-35 元 | >35 元 |
| **家居家装** | ≤8 元 | 8-13 元 | 13-20 元 | 20-28 元 | >28 元 |
| **母婴用品** | ≤10 元 | 10-16 元 | 16-24 元 | 24-32 元 | >32 元 |

**填写建议**：
- 首次合作：标准放宽 20%，给达人磨合空间
- 二次合作：按标准执行，达标追加，不达标止损
- 头部达人：可适当放宽（品牌曝光价值）

## 四、常见填写错误与避坑指南

### ❌ 错误 1：人群定义过于宽泛

**错误示范**：
```
年龄：18-45 岁
性别：不限
地域：全国
职业：不限
```

**问题**：等于没说，达人无法针对性创作内容

**正确做法**：
```
年龄：25-32 岁（核心 28-30 岁）
性别：女性占比 70-80%
地域：新一线/二线城市（成都、杭州、武汉）
职业：职场新人至资深白领（工作 1-8 年）
```

### ❌ 错误 2：卖点太多，重点不突出

**错误示范**：
```
核心卖点：
1. 控油  2. 去屑  3. 柔顺  4. 留香  5. 修复  6. 防脱  7. 增亮...
```

**问题**：达人记不住，用户记不住，传播效率低

**正确做法**：
```
核心卖点（最多 3 个）：
1. 控油 72 小时（专利成分，实验室数据支撑）
2. 去屑止痒（RG0 认证，敏感头皮可用）
3. 持久留香（大牌同款香调，留香 48 小时）

辅助卖点（可选）：
- 修复受损发质
- 提升光泽度
```

### ❌ 错误 3：内容要求模糊

**错误示范**：
```
内容要求：视频要有趣，能吸引人，有创意
```

**问题**：主观描述，无法执行和验收

**正确做法**：
```
内容要求：
1. 前 30 秒必须出现产品核心卖点（至少 3 个参数）
2. 视频中必须有对比环节（新旧/开关/模式对比）
3. 视频中必须展示真实使用场景（非棚拍）
4. 结尾必须有购买引导（口播或字幕提示"点击左下角"）
5. 评论区必须置顶产品信息（参数表 + 购买链接 + 优惠政策）
```

### ❌ 错误 4：验收标准缺失

**错误示范**：
```
验收标准：效果好就追加合作
```

**问题**：什么是"好"？没有量化标准

**正确做法**：
```
验收标准：
- CPUV ≤ 10 元：立即追加，签订长期合作
- CPUV 10-15 元：观察至 15 天，再决定
- CPUV 15-20 元：不再追加，自然长尾
- CPUV > 20 元：止损，复盘原因
```

## 五、填写完成后自检清单

### 完整性检查（权重 40%）

- [ ] 所有必填项已填写（标有*的字段）
- [ ] 第三部分"目标人群定义"完成度≥90%
- [ ] 第五部分"内容要求"有具体的结构模板
- [ ] 第八部分"验收标准"有量化指标
- [ ] 附件材料已准备齐全

### 清晰度检查（权重 30%）

- [ ] 人群画像具体到年龄段、城市、职业（不是宽泛的范围）
- [ ] 核心卖点不超过 3 个，且有数据/证据支撑
- [ ] 内容要求可执行、可验收（不是主观描述）
- [ ] 验收标准量化（有具体数字，不是"效果好"）

### 一致性检查（权重 30%）

- [ ] 人群画像与产品价位匹配（高端产品 vs 高消费人群）
- [ ] 内容形式与人群偏好匹配（理性人群 vs 测评内容）
- [ ] 达人人设与品牌形象匹配（专业品牌 vs 专业测评达人）
- [ ] 投放目标与 KPI 匹配（品牌建设 vs 曝光量，种草引流 vs 进店数）

**自评得分 ≥ 80 分即可提交，<80 分建议重新完善。**

## 六、提交后的流程

### 6.1 服务商工作流程

```
Day 1-2：服务商消化 Brief，确认理解无误
Day 3-5：基于 Brief 筛选达人，提供候选人名单（5-10 位）
Day 6-8：商家确认达人，服务商对接档期和报价
Day 9-11：达人创作脚本，商家审核并提出修改意见
Day 12-14：达人拍摄剪辑，商家审片并确认发布
Day 15：视频发布，进入数据监控期
Day 16-22：数据监控（Day3/Day7 关键节点）
Day 23-30：项目复盘，决定是否追加合作
```

### 6.2 商家配合事项

**脚本审核阶段**：
- 重点检查：核心卖点是否完整传达、有无违规风险、人群匹配度
- 修改意见：一次性提出，避免反复修改（最多 2 轮）

**视频审片阶段**：
- 重点检查：画面质量、字幕准确性、购买链接有效性
- 修改权限：仅可要求修改硬伤（错别字、链接错误等），不可要求大幅改动

**数据监控阶段**：
- Day3：关注初步 CPUV 和评论风向
- Day7：评估是否值得追加投放
- Day15：最终复盘，沉淀经验

## 八、常见问题 Q&A

### Q1：我是新品，没有用户数据，如何填写人群画像？

**A**：参考以下方法：
1. **竞品对标**：找到天猫同类目 TOP3 品牌，研究他们的用户画像
2. **问卷调研**：投放前做小范围问卷（100-200 份），了解潜在用户需求
3. **焦点小组**：邀请 5-8 位目标用户深度访谈，挖掘深层动机
4. **达人粉丝画像**：查看意向合作达人的粉丝画像数据

### Q2：预算有限，如何分配达人类型？

**A**：建议采用"721 原则"：
- 70% 预算：成熟型达人（历史表现稳定，风险低）
- 20% 预算：潜力型达人（有新意，可能爆）
- 10% 预算：测试型达人（新类型，探索可能性）

### Q3：达人不愿意按 Brief 执行怎么办？

**A**：分级处理：
- **原则性问题**（卖点遗漏、违规风险）：坚持要求修改
- **非原则性问题**（表达方式、创意发挥）：尊重达人专业性
- **预防胜于治疗**：在合作合同中明确修改权限和验收标准

### Q4：如何判断 Brief 填写质量？

**A**：让服务商/达人阅读 Brief 后，请他们复述：
- 目标人群是谁？（能否描述出具体画像）
- 核心卖点是什么？（能否说出 3 个关键点）
- 内容要求有哪些？（能否列出必须包含的要素）
- 验收标准是什么？（能否说出 CPUV 分级）

如果对方能准确复述，说明 Brief 清晰；如果对方一脸茫然，说明需要重新完善。
