# 雷士照明 KOL 投放执行 SOP（2026 版）

## 一、达人筛选阶段

### 1.1 初步筛选标准

**硬性指标**：
- [ ] 粉丝数 3-50 万（腰部达人优先）
- [ ] 近 30 天更新频率≥4 条/月
- [ ] 单视频平均播放量≥2 万
- [ ] 评论率≥0.3%（评论数/播放量）
- [ ] 无重大负面舆情

**粉丝画像匹配度**：
- [ ] 25-45 岁用户占比≥60%
- [ ] 房主/准房主/家长群体占比≥30%
- [ ] 男性/女性比例符合产品定位（科技类男多，母婴类女多）

**内容风格评估**：
- [ ] 历史内容有家居/数码/生活好物推荐经验
- [ ] 视频质量精良（画面清晰、剪辑流畅）
- [ ] 人设清晰（专业测评/真实分享/生活记录）
- [ ] 评论区互动良好（UP 主回复及时）

### 1.2 背调清单

**数据真实性核查**：
```bash
# 使用第三方工具查询（如飞瓜数据、蝉妈妈等）
- [ ] 粉丝增长曲线是否正常（排除买粉）
- [ ] 播放量/点赞比是否合理（正常范围 100:1-300:1）
- [ ] 评论内容是否真实（排除水军）
```

**商业合作历史**：
- [ ] 过往合作品牌调性（避免低端品牌）
- [ ] 广告视频占比（不超过 30% 为佳）
- [ ] 广告视频互动数据 vs 日常视频（差距<50% 为佳）

**联系方式建立**：
- [ ] 获取商务微信/邮箱
- [ ] 确认报价（刊例价 vs 实际成交价）
- [ ] 了解档期情况（通常需提前 2-3 周预定）

### 2.2 内容要求（硬性规定）

**必须包含的要素**（缺少任一不予通过）：
- [ ] **前 30 秒出现至少 3 个核心参数**（Ra 值、频闪、色温等）
- [ ] **至少 1 组对比展示**（新旧灯具/开灯关灯/不同模式）
- [ ] **至少 2 个真实使用场景**（孩子写作业、成人阅读等）
- [ ] **结尾购买引导**（口播或字幕提示"点击左下角"）

## 四、数据监控阶段

### 4.1 关键指标定义

**CPUV（Cost Per Unique Visitor）**：
```
CPUV = 消耗金额 ÷ 15 日进店总数
```
- 数据来源：阿里 V 任务后台 + 生意参谋
- 统计周期：视频发布后 15 天
- 行业基准：照明灯具类目 CPUV 约 15-25 元

**进店率**：
```
进店率 = 进店人数 ÷ 视频播放量 × 100%
```
- 健康范围：1%-2%
- 低于 0.5% 需优化购买引导

**转化率**：
```
转化率 = 成交订单数 ÷ 进店人数 × 100%
```
- 健康范围：1.5%-3%
- 低于 1% 需优化详情页或价格策略

### 4.2 监控时间节点

**Day 1（发布当天）**：
- [ ] 播放量增长曲线（每小时记录）
- [ ] 互动数据（点赞、硬币、收藏、评论）
- [ ] 评论情感分析（正面/中性/负面占比）
- [ ] 异常数据预警（如播放量停滞、负面评论激增）

**Day 3（关键节点）**：
- [ ] 计算初步 CPUV（基于前 3 天进店数据）
- [ ] 如 CPUV>25 元，立即优化评论区话术
- [ ] 如播放量<预期 50%，考虑 DOU+ 助推
- [ ] 记录典型用户评价（用于后续宣传）

**Day 7（中期评估）**：
- [ ] 计算完整 CPUV（7 天进店数据）
- [ ] 评估是否值得追加投放
  - CPUV<10 元 → 立即追加，签订长期合作
  - CPUV 10-15 元 → 观察至 15 天，再决定
  - CPUV 15-20 元 → 不再追加，自然长尾
  - CPUV>20 元 → 止损，复盘原因
- [ ] 负面评论占比检查（如>15%，启动公关预案）

**Day 15（最终复盘）**：
- [ ] 最终 CPUV 计算
- [ ] ROI 核算（GMV ÷ 投放成本）
- [ ] 达人表现评级（S/A/B/C/D）
- [ ] 决定是否二次合作
- [ ] 案例归档（成功/失败经验沉淀）

### 4.3 预警机制

**黄色预警（需关注）**：
- CPUV 连续 3 天>20 元
- 负面评论占比>10%
- 播放量增长明显放缓（日增幅<10%）

**橙色预警（需干预）**：
- CPUV 连续 3 天>30 元
- 负面评论占比>15%
- 出现恶意差评或竞品攻击

**红色预警（需止损）**：
- CPUV>50 元
- 负面评论占比>30%
- 出现严重舆情危机（产品质量质疑、虚假宣传投诉等）

**应对措施**：
```markdown
黄色预警：
- 优化评论区话术，加强正面引导
- 追加优质评论点赞，提升可见度
- 联系 UP 主配合回复关键问题

橙色预警：
- 暂停追加投放预算
- 紧急公关预案启动
- 与 UP 主沟通删除/修改争议内容

红色预警：
- 立即停止合作
- 法务介入评估风险
- 官方声明准备（如需）
```
